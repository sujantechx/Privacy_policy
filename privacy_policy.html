5. Data Storage and Security

5.1 Where We Store Your Data

Cloud Infrastructure:

  • Primary data storage: Firebase Cloud Firestore (Google Cloud Platform)
  • File storage: Firebase Storage
  • Authentication: Firebase Authentication
  • Located in secure data centers with industry-standard security

Local Device Storage:

  • Shared Preferences: User preferences, session tokens
  • Local cache: Temporary app data for offline functionality

5.2 Security Measures

We implement appropriate technical and organizational measures to protect your data:

  • Encryption: Data transmitted over HTTPS/TLS encryption
  • Authentication: Secure Firebase Authentication with multi-factor support
  • Access Controls: Role-based access permissions (User, Worker, Admin)
  • Monitoring: Firebase Crashlytics for error tracking and security monitoring
  • Regular Updates: Security patches and updates
  • Data Minimization: We collect only necessary information

5.3 Data Retention

  • Active Accounts: Data retained while your account is active
  • Inactive Accounts: Data may be deleted after extended inactivity (subject to legal requirements)
  • Booking Records: Retained for 7 years for tax and legal compliance
  • Verification Documents: Retained as long as legally required
  • Payment Records: Retained per financial regulations

Deletion Requests: You may request account deletion at any time (see Section 8).
